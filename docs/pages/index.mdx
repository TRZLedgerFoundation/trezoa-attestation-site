## Manual Installation

To manually add the TAS library to your trezoa, you can use the following commands:

### Typescript / Javascript

```bash [npm]
npm i tas-lib 
```

```bash [yarn]
yarn add tas-lib 
```

```bash [pnpm]
pnpm i tas-lib 
```

### Rust

```bash
cargo add trezoa-attestation-service-client
```
